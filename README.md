## 🎄 프로젝트 트리

```
src
 ┣ assets       // svg 파일
 ┣ components   // 공통으로 사용하는 컴포넌트
 ┣ hooks        // Custom Hooks
 ┣ pages        // 페이지
 ┣ routes       // Route
 ┣ services     // API 호출 관련
 ┣ states       // 전역 상태관리
 ┣ styles       // 전역 style
 ┣ types        // 전역 type 정의
 ┣ utils        // 전역 util 함수 정의
```
